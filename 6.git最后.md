## 4. 忽略文件

先

`vim .gitignore`

创建文件

在这个文件里头操作

```
还可以写自己
.gitignore

*.h // 只要以这个结尾的
!a.h  // 该文件排除
files/  // 会忽略该文件夹霞的所有文件
```





应用场景：只推送代码，不推送数据库文件

​			敏感数据不能推送到远程
